# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Provide technical feedback focused on code and behavior.
- Be open to different viewpoints.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Discrimination of any kind.
- Trolling, intimidation, or abusive language.
- Publishing private information without consent.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and other official communication channels.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.
They may remove, edit, or reject comments, commits, code, issues, and other contributions not aligned with this policy.

## Reporting

Report unacceptable behavior privately to the maintainer via the security/contact channel defined in `SECURITY.md`.
Primary contact details are listed in `SUPPORT.md`.
